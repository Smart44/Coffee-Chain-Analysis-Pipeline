# Coffee-Chain-Analysis-Pipeline

This project is an automated data pipeline designed to:

    Extract sales data from a raw CSV file.

    Clean and Transform the data, then load it into a PostgreSQL database.

    Execute SQL queries to perform business analysis, such as profit calculations.

    Visualize the findings by automatically generating and saving charts using Seaborn.

Technologies Used

    Python (Pandas, Seaborn, SQLAlchemy)

    PostgreSQL (Relational Database Management)

    DBeaver (Database administration and SQL development)
